Data We Collect From Visitors

We use Cloudflare Web Analytics, which: - Is privacy-preserving by design - Does not use cookies - Does not track users across sites - Does not collect personal information - Provides aggregate metrics only - Complies with privacy regulations Analytics Opt-Out Options: - Use standard browser privacy features (they’re respected) - Enable DNS-level blocking - Use script-blocking extensions

We explicitly do not: - Store personal data - Use tracking cookies - Maintain user accounts - Process sensitive information - Track individual users - Store IP addresses - Use third-party analytics services

Cloudflare provides our infrastructure services including content delivery, DDoS protection, DNS resolution, and privacy-preserving web analytics. For details about their data handling, see Cloudflare’s Privacy Policy.
