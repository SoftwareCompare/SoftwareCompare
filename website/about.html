What is Software Compare?

It is hard to find simple and objective data on software online, and this may cause confusion for someone in search of software to use

Software Compare aims to provide people tools to choose software that best suits their needs. The charts compare an assortment of criteria, but users access their own needs and decide which software suits them best.
