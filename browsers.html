Open Source {% include link text="Open Source Software" link="https://www.ibm.com/topics/open-source" %} refers to computer programs that make their code freely available to the public. This allows for the code to be analyzed for any security flaws and malicious code. The code can also be "forked" and turned into a new application if anyone wishes to do so.
Default Global Privacy Control {% include link text="Global Privacy Control" link="https://globalprivacycontrol.org/faq" %} is a specification that can be toggled as a setting in some browsers that tells websites if the user wants their information to be sold or shared.
