Non-Monetary ways to help

Software Compare takes a lot of effort to maintain and update, so we appreciate any help you can offer!

Some ways to help are:

  • Fact check the site and open a {% include link text="Github issue" link="https://github.com/SoftwareCompare/SoftwareCompare/issues" %} or {% include link text="pull request" link="https://github.com/SoftwareCompare/SoftwareCompare/pulls" %} so we can ensure accuracy of our information.
  • Join our {% include link text="Matrix" link="https://matrix.to/#/#softwarecompare:matrix.org" %}, {% include link text="Discord" link="https://discord.gg/SvZXsWaT2w" %} or other socials to help answer questions and interact with the community.
  • If you are a graphic designer or programmer, and see anything that can be simplified or improved, reach out and we can discuss further.
